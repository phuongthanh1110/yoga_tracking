## Build & Run

Prerequisites:
- Flutter SDK installed and added to your PATH
- Platform toolchains set up (Xcode for iOS/macOS, Android Studio/SDK for Android, Chrome for web)

Steps:
1) Install dependencies:
```
flutter pub get
```
2) Run on a connected device/emulator (picks the first available device):
```
flutter run
```
3) Choose a specific target (examples):
- Android emulator/device: `flutter run -d android`
- iOS simulator: `flutter run -d ios`
- macOS desktop: `flutter run -d macos`
- Web (Chrome): `flutter run -d chrome`

Build artifacts:
- Android APK: `flutter build apk`
- iOS: `flutter build ios` (requires Xcode and a signing setup)
- Web: `flutter build web`
